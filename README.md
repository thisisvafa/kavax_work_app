# Base App

## Installation

-   `composer install`
-   `npm install`
-   `cp .env.example .env`
-   `php artisan key:generate`
-   `php artisan jwt:secret`
-   `php artisan migrate`
-   `php artisan db:seed`
-   `php artisan serve`
-   The application should be served in 127.0.0.1:8000 according to port availabilty
